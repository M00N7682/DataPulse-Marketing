광고산업에서 데이터 기반 의사결정은 핵심 경쟁요소로 부상하였으나, 실제 광고성과 로그 데이터는 개인정보보호 규제로 접근이 어렵고, 캠페인별 데이터 편중이 심해 예측모델의 일반화 성능이 떨어진다. 본 연구는 이러한 데이터 제약 환경에서 **합성 데이터 생성 기반의 하이브리드 학습 프레임워크**를 통해 마케팅 성과 예측의 정확도와 효율성을 개선하는 실용적 솔루션을 제안한다.

8,000개 고객의 디지털 마케팅 캠페인 데이터를 기반으로 CTGAN과 TabDDPM을 활용하여 17개 핵심 피처의 고품질 합성 데이터를 생성하였다. 생성된 합성 데이터는 Kolmogorov-Smirnov Distance(0.074~0.082), 상관관계 보존도(0.923~0.935) 등 모든 통계적 품질 지표에서 임계값을 통과하여 실제 데이터와 높은 유사성을 보였다.

하이브리드 학습 실험을 통해 실제 데이터만 사용한 기준 모델(F1-Score: 0.8906) 대비 실제 데이터와 합성 데이터를 50:50으로 혼합한 시나리오에서 2.5% 성능 향상(F1-Score: 0.9125)을 달성하였다. 특히 데이터가 부족한 콜드 스타트 상황에서는 기준선 대비 21% 성능 개선을 보여, 신규 캠페인 런칭 시 합성 데이터의 효과적 활용 가능성을 실증하였다.

본 연구는 개인정보보호 규제가 강화되는 환경에서도 합성 데이터를 통해 마케팅 예측 모델의 성능을 효과적으로 향상시킬 수 있음을 입증하며, 데이터 접근이 제한적인 마케팅 의사결정 상황에서 비용 효율적이고 실용적인 대안을 제시한다.
